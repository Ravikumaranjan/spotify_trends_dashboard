 🎵 Spotify Performance Analytics Dashboard

An interactive **Power BI dashboard** that analyzes the **Spotify Top 50 Global** dataset to uncover insights into song popularity, artist performance, album trends, and streaming patterns through visually appealing and interactive reports.

 Overview

This project demonstrates how **Microsoft Power BI** can transform raw music streaming data into actionable business insights. The dashboard provides comprehensive analysis of Spotify's Top 50 Global songs using interactive visualizations, KPI cards, DAX measures, and trend analysis.

Whether you're a music enthusiast, business analyst, or data analyst, this dashboard helps understand global music trends and artist performance.

 Features

- 📊 Interactive KPI Cards
- 🎤 Artist Performance Analysis
- 🎵 Top Songs Analysis
- 💿 Album Insights
- 📅 Release Trend Analysis
- 📈 Popularity Trend Visualization
- 🔍 Dynamic Filtering with Slicers
- 📉 Forecasting & Trend Analysis
- 🎨 Spotify-inspired Dashboard Theme

- Album Type
